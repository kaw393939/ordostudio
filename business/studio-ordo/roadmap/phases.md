# Studio Ordo — Roadmap Phases (Platform)

**Owner:** Keith Williams · **Last updated:** 2025-07-14

These phases are intended to become sprint packages. Phases are ordered by
revenue-enabling priority and persona-journey dependencies.

---

## Phase 1 — Managed marketplace core
- Apprentice profiles + directory
- Intake → Deal model
- Maestro assignment + approval gates
- Upfront payment capture (Stripe)
- Refund tooling (admin)
- Ledger generation (earned) + approval flows

## Phase 2 — Payouts + commissions
- Stripe Connect onboarding (Express)
- Provider payout execution
- Referrer commission payouts
- Payout dashboards + exports

## Phase 3 — Affiliate & referral system *(NEW)*
*Unblocks: Sophia Rivera (Affiliate) persona; flywheel growth engine*
- Affiliate registration + W-9 collection
- Referral link / QR code generation with attribution
- Attribution cookie tracking (90-day window, first-touch)
- Commission ledger at 25% lifetime rate (EARNED → APPROVED → PAID)
- Affiliate dashboard (referrals, commissions, payouts)
- Co-branded asset generation

## Phase 4 — Apprenticeship journey *(NEW)*
*Unblocks: Alex Torres (Apprentice) persona; core differentiator*
- Apprentice application + acceptance workflow
- Level progression tracking (1–4)
- Gate project submission + maestro rubric scoring
- Spell Book + Context Pack progress tracking
- Demo Day management
- Career Bridge profile generation

## Phase 5 — Community & assessments *(NEW)*
*Unblocks: Maria Santos (Community), Marcus Chen & Lisa Okafor (Assessment)*
- EverydayAI event management + attendance tracking
- Town Hall registration + recording
- Human Edge Scorecard (interactive web → PDF)
- Team Readiness Assessment (32-point pre/post)
- Lead magnet delivery with email capture

## Phase 6 — Content access tiers *(NEW)*
*Unblocks: subscription revenue; content monetization*
- Subscription model (Free → AI-Assisted → Human+AI → Dedicated)
- Content paywall / gating by tier
- Stripe recurring billing
- Tier management UX

## Phase 7 — Delivery OS
- Deal artifacts + follow-up actions
- Client comms thread (or structured notes)
- Completion criteria per offer
- Quality signals + escalation
- Engagement progress tracking
- Pre/post assessment comparison reports

## Phase 8 — Newsletter + communications
- Subscribers + unsubscribe
- Scheduled sends + delivery logs
- Field report → newsletter content pipeline
- Editorial dashboards
- Content repurposing engine

## Phase 9 — Career Bridge *(NEW)*
*Unblocks: James Park (Hiring Manager) persona; placement revenue*
- Employer registration + network tiers
- Candidate profiles from apprentice portfolios
- Job matching + placement tracking
- Placement fee invoicing + 90-day guarantee

## Phase 10 — Data ingestion + intelligence
- Ingestion services (meetup/eventbrite, tech sites, youtube transcripts)

## Phase 11 — MCP + automation
