# Studio Ordo — Security & Privacy Baseline

## Security goals
- Protect user accounts and admin operations.
- Protect sensitive client information.
- Ensure money workflow is auditable and safe.

## Baseline controls (MVP)
- Role-based access control.
- CSRF protections for mutations.
- Audit logging for mutations.
- Least-privilege tokens for automation.

## Data handling
- Collect minimal PII.
- Store only what is needed for delivery and compliance.

## Privacy disclosures
- Referral attribution uses a cookie.
- Newsletter subscription uses email.

## Incident response (outline)
- Detect → contain → remediate → communicate → postmortem.
