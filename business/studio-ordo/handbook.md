# Studio Ordo — Operating Handbook (Fortune-100 Style, Small-Team Edition)

This handbook captures how Studio Ordo operates as a **managed marketplace + apprenticeship + editorial engine**.

Principles:

- **Anti-hype**: proof over vibes.
- **Standardized offers**: union pricing, predictable scope.
- **Adult supervision**: maestro gatekeeping protects clients and apprentices.
- **Auditable operations**: every mutation is logged; money movement is ledgered.

## What this is

- A professional baseline for Keith Williams' LLC to run Studio Ordo.
- A source document for converting strategy → epics → sprint packages.
