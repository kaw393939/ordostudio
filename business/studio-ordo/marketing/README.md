## Voice Requirements

All marketing copy follows the Studio Ordo brand guide:
- **Anti-hype:** Every claim backed by data or artifact
- **Swiss precision:** Clean, direct, no filler words
- **No fake scarcity, no dark patterns**
- **CTA verb system:** View training tracks / Book a technical consult / Join the studio / Get the Ordo templates

## Usage

These are copy specifications — the content ready for implementation in the LMS system sprints. No additional writing is needed. System sprints implement these directly into Next.js pages.
